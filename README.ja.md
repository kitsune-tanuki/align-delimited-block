# Align Delimited Block

[English README / 英語版はこちら](./README.md)

カーソル周辺の区切り文字付きテキストブロックを自動整列する VSCode 拡張です。

多くの整列ツールとは異なり、この拡張は **表示幅 (display width)** を計算して整列します。そのため、日本語・中国語・韓国語などの全角文字や絵文字を含むテキストでも正しく揃います。

AsciiDocテーブル、LaTeX align環境、アロー式、設定ファイルなどに便利です。

---

## 特徴

* デフォルト区切り文字 (設定可能) で整列 (初期値: `|`)
* 任意区切り文字で整列可能
* 複数文字区切り対応 (`=>`, `::`, `->` など)
* カーソル周辺の連続行を自動検出
* 1行のみのブロックは整列をスキップ
* 選択不要
* 整列結果に変化がない場合は編集をスキップ
* 軽量・高速
* 全角文字 (CJK) 対応
* 絵文字や結合文字に対応
* 文字数ではなく表示幅で整列
* 検証処理とエラーハンドリングを強化

---

## この拡張を作った理由

多くの整列ツールは文字数で整列するため、日本語などを含むと見た目が崩れます。

例:

一般的なフォーマッターの出力:

```text
a   | b
日本語 | c
```

期待される整列:

```text
a      | b
日本語 | c
```

この拡張は表示幅を計算することで、この問題を解決します。

---

## 使い方

区切り構造の中にカーソルを置きます。

前後の連続行を自動検出して整列します。

検出されたブロックが1行のみの場合は、整列をスキップします。

整列結果が元のテキストと同一の場合は、編集を適用しません。

---

## 例

> [!NOTE]
> 視認性を高めるため、以下の例ではスペースを `·` (MIDDLE DOT) で表記しています。
> 実際の出力では通常のスペースが使用されます。

### パイプ区切り

Before:

```text
a·|·bb·|·c
aaa·|·b·|·cc
```

Run:

```text
Align Delimited Block (Default Delimiter)
```

Result:

```text
a···|·bb·|·c
aaa·|·b··|·cc
```

---

### カスタム区切り

Before:

```text
x·=>·10
long_variable·=>·20
y·=>·5
```

Run:

```text
Align Delimited Block (Custom Delimiter)
```

Enter:

```text
=>
```

Result:

```text
x·············=>·10
long_variable·=>·20
y·············=>·5
```

---

### CJKの例

Before:

```text
name·|·value
長い名前·|·10
x·|·5
```

Result:

```text
name·····|·value
長い名前·|·10
x········|·5
```

---

### LaTeX align環境の例

Before:

```text
a·&·=·b·+·c
long·&·=·x
z·&·=·100
```

Delimiter:

```text
&
```

Result:

```text
a····&·=·b·+·c
long·&·=·x
z····&·=·100
```

## コマンド

| コマンド                                  | 説明                                 |
| ----------------------------------------- | ------------------------------------ |
| Align Delimited Block (Default Delimiter) | 設定されたデフォルト区切り文字で整列 |
| Align Delimited Block (Custom Delimiter)  | 任意の区切り文字で整列               |

---

## キーバインド

| キー         | 動作                       |
| ------------ | -------------------------- |
| Ctrl+Shift+A | デフォルト区切り文字で整列 |
| Ctrl+Alt+A   | 任意の区切り文字で整列     |

---

## 用途

* AsciiDocのテーブル
* LaTeX align環境
* Markdownのテーブル
* 設定ファイル
* カラム整形テキスト
* アロー式
* 区切りテキスト全般

---

## 必要条件

なし

---

## 拡張設定

この拡張には以下の設定があります:

* `alignDelimitedBlock.defaultDelimiter`  
  `Ctrl+Shift+A` 実行時に使用されるデフォルト区切り文字を設定します。

初期値: `|`

---

## 補足

* 1行のみのブロックは意図的にスキップされます
* 整列しても結果が変わらない場合は編集されません
* 空文字や不正な区切り文字は安全に拒否されます
